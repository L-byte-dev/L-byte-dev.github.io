Proyecto Java

El objetivo de este proyecto fue crear una aplicación que simule un sistema bancario. Un cliente del banco puede tener múltiples cuentas. Los clientes pueden gestionar sus cuentas y consultar el saldo total. Las cuentas bancarias permiten depositar, retirar y consultar saldo. Hay dos tipos de cuentas: de ahorro, que generan intereses, y corriente, con posibilidad de sobregiro. El banco permitiende registrar nuevos clientes y abrir cuentas.

Proyecto Spring
